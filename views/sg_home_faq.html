Для подключения

  • скачаь последнюю версию StyleGuide
  • подключить файл стилей style_guide.css к проекту
  • переместить изображения в папку проекта в директорию /images
  • переместить шрифты в папку проекта в директорию /fonts

Для подключения через NPM

Для подключения к проекту используется локальный репозитории, поэтому необходимо выполнить следующие дейсвия:

В качестве внутреннего репозитория npm-пакетов используется Nexus (https://echo.centre-it.com/nexus/).

Настройка Nexus для проксирования npm-репозиториев произведена в соответствии с документацией. Создан репозиторий npmjs Registry (проксирующий registry.npmjs.org), репозиторий npm-internal для регистрации собственных компонентов, а также группа npm-all, объединяющая оба репозитория.

Чтобы утилита npm использовала внутренний репозиторий, необходимо создать файл .npmrc в корневом каталоге пользователя:

  • В Linux, OS X — ~/.npmrc
  • В Windows — c:\Users\\.npmrc

Пример содержимого файла: registry=https://echo.centre-it.com/nexus/content/groups/npm-all/ always-auth=true strict-ssl=false email=Steve.Jobs@centre-it.com _auth=YWRtaW46YWRtaW4xMjM=

В значение email необходимо подставить свой рабочий адрес электронной почты, в значение _auth — строку вида логин:пароль, преобразованную в base64. Её можно получить следующим образом:

В Linux, OS X:

echo -n 'user:password' | openssl base64

В Windows (в терминале cmd):

@powershell -NoProfile -ExecutionPolicy Bypass -Command "System.Convert::ToBase64String([System.Text.Encoding]::UTF8.GetBytes(\"user:password\"))"
