Сведения о корректирующем действии

  • Тип отчета производителя о корректирующем действии по безопасности медицинского изделия
    первоначальный отчет
  • Номер отчета о корректирующем действии
    234а43-4
  • Дата отчета о корректирующем действии
    15.06.2009
  • Общие сведения и причина корректирующих действий
    По информации, полученной от производителя, по причине отклонения в работе конденсатора низкого напряжения, батарея в указанных изделиях может разряжаться с более высокой скоростью, нежели это предусмотрено
  • Описание и обоснование корректирующих действий
    Проведение компании по обновлению программного обеспечения устройств программирования, записи и наблюдения (PRM)
  • Рекомендации для поставщиков и пользователей
    Пациентам, которым имплантированы данные медицинские изделия, рекомендовано назначить посещение клиники для обновления устройства с помощью нового программного обеспечения
  • Процесс реализации корректирующих действий
    Назначение пользователям замены оборудования
  • Сроки реализации корректирующих действий
    15.06.2009 - 15.06.2010
